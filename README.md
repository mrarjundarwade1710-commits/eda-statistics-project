# Advanced Exploratory Data Analysis & Statistical Analysis using Python

## Project Overview
This project performs exploratory data analysis and statistical analysis on a dataset using Python.

The goal is to identify patterns, correlations, distributions, and insights using statistical methods and data visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Analysis Steps
1. Data loading and preprocessing
2. Data cleaning
3. Statistical analysis
4. Correlation analysis
5. Distribution analysis
6. Data visualization

## Statistical Techniques Used
- Mean
- Median
- Variance
- Standard Deviation
- Skewness
- Kurtosis
- Probability distributions

## Visualizations
- Scatter plots
- Distribution plots
- Correlation heatmaps
- Boxplots

- Add hypothesis testing
